# proyecto-final_pension65-jovenes
Proyecto para el curso de Inferencia Causal. Se una propuesta de investigación con un análisis de la base de datos de la ENAHO 2024 y pruebas de regresión. El tema general es el efecto de que un adulto mayor en el hogar sea beneficiario de Pensión 65 en la oferta laboral de los jóvenes.
